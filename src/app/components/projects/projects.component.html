COVID-19 Tracker

Unfortunately, we have seen the biggest pandemic in human era and that did messes up a lot of people's lives. It is a hard time for everyone as we have to check the number of cases every signle day and hoping for it to end and I used to check stats of COVID-19 on bing.com/covid and I was very curious how did it worked, so I started building my own and utilized the unfortunate time. Started from very scratch by creating it's design in a piece of paper and then drew it in figma software called draw.io. Later, I started building it and displayed the Map using the Google Maps API and fetched the covid data from an API("https://disease.sh/v2/historical/all?lastdays=150"). I used chart.js library to display beautiful looking charts.
SIDE NOTE: If you are seeing this after few years, I am glad you made through the pandemic in 2020.

Expense Tracker App

Expense tracker application can store the income and the expenses. User can register and Sign In the application. User will be able to record expenses on each day he makes. User can update and delete his expenses and user can also change his account details like email, password and income.
Reports can be plotted as various graphs to identify on which the expense is more. It can show expenses by category and also by a particular date. User can also check his Expense VS Income graph. Also, on logging into the system Dashboard have an indicator of all the expenses and also the one which is peaking. It can give the graph of the expenses across a year.
